Random
===

A text-based rpg named _Random_.

Requirements
===
* `Java 8`
* `Maven`

To play the game run the following code in terminal:
    
 ```
 mvn package
    
 mvn exec:java -Dexec.mainClass="com.magyart.random.Main" 
 ```
